The app being enhanced

A self-contained, offline-installable music practice PWA - vanilla JS, zero build step, zero frameworks, 158 commits of compound iteration. The north star: an adaptive tutor for soloing and songwriting confidence.

Compose

Key/mode-aware progression builder: pick E Mixolydian and the in-key palette leads with the right seven chords, roman-numeral labeled. Root change transposes the built progression; mode change re-harmonizes it. Famous-progression starters and next-chord suggestions.

Solo + Studio

One tap carries the built key/mode to backing-track soloing: matched tracks, the solo scale on a real fretboard, position-walking up the neck. Theory taught just-in-time, never as a wall.

Library + Setlist

Repertoire with chord sheets, key-first tags, capo badges; a running-order Set with edit-mode-gated reorder/remove and persistent undo (a prior swarm's human-factors work).

Tune + Instruments

Mic autocorrelation tuner with stability smoothing. 8 instrument profiles: guitar (standard, drop-D, open-G), ukulele, banjo, mandolin, mandola, cigar box - every diagram re-renders per instrument.
